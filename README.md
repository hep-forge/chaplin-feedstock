Feedstock for [chaplin](https://chaplin.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

CHAPLIN - Complex Harmonic Polylogarithms in FORTRAN
